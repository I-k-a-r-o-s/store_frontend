## Setup and Usage Instructions

This is a frontend application for a store management system built with React, TypeScript, and Vite. It allows users to view, create, update, and delete products.

### Prerequisites

- Clone and set up `https://github.com/I-k-a-r-o-s/store_backend`

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/I-k-a-r-o-s/store_frontend.git
   cd store_frontend
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Create a .env file and use `VITE_BASE_URL=http://localhost:5000/api` as the base URL.

### Running the Application

1. Start the development server:

   ```bash
   npm run dev
   ```

2. Open your browser and navigate to `http://localhost:5173` (or the URL shown in the terminal).
